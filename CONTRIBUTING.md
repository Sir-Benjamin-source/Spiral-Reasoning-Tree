# Contributing to Spiral Reasoning Tree

Thank you for considering contributing!

**Ways to Help**
- Refine the core formula or add new resonance variants
- Improve visualizations (2D/3D/4D)
- Add more examples or benchmarks
- Write tests
- Fix bugs or add features (e.g., adaptive parameters)

**Guidelines**
- Keep changes modular (core untouched if possible)
- Reference Zenodo v1.1 for theory alignment
- Use lowercase directories
- Add/update tests when adding features

**Process**
1. Fork the repo
2. Create a feature branch (`git checkout -b feature/amazing-thing`)
3. Commit (`git commit -m 'Add amazing thing'`)
4. Push & open a Pull Request
